> Start Apache and MySQL services of Laragon
> Open Laragon Terminal with the project path from: D:\laragon\www\lara-collab and run: npm run dev
> In other Laragon Terminal tab, run: php artisan serve (optional)
> App URL: http://127.0.0.1:8000

> Database: http://localhost/phpmyadmin > root, empty password
> In other Laragon Terminal tab, you can run commands like: php artisan migrate..
